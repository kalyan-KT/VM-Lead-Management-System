# Lead Management System

A personal lead management system for zero-miss follow-ups. Track LinkedIn leads, schedule actions, and never forget a follow-up.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```

3. Build for production:
   ```sh
   npm run build
   ```
